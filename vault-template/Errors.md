# Errors
Document recurring errors and proven fixes.

## Template
- **Error:** 
- **Cause:** 
- **Fix:** 
- **Context:** 
- **Verified:** 

Example:
- **Error:** `Module not found`
- **Cause:** dependency installed in wrong environment
- **Fix:** use the project virtualenv before installing
- **Context:** Python projects with multiple interpreters
- **Verified:** `python -m pytest`
